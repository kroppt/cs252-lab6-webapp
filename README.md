# cs252-lab6-webapp
Implementation of a web application using HTML5 and cloud services

### Prerequisites

[golang](https://golang.org/project/)

### Running

Fetch the code
```
git clone https://github.com/kroppt/cs252-lab6-webapp
cd cs252-lab6-webapp
```

Run the code:

```
go run webapp.go
```
or
 
```bash
go build webapp.go
./webapp
```
